# ISESASS Frontend Setup Requirements
# This file contains the essential setup steps for the project

# 1. Node.js Requirements
# - Node.js version 16.0.0 or higher
# - npm version 8.0.0 or higher

# 2. Global Dependencies (Auto-installed by setup scripts)
# - Vercel CLI: npm install -g vercel
# - kill-port utility: npm install -g kill-port

# 3. Environment Setup
# Create .env.local file with:
REACT_APP_API_URL="Contact teammate"

# 4. Dependencies to install
# Run: npm install
# This will install all dependencies from package.json

# 5. Start the development server
# Run: npm start

# 6. Access the application
# Open: http://localhost:3000

# 7. Key Dependencies (Auto-installed with npm install)
# - React 19.1.1
# - React DOM 19.1.1
# - TypeScript 4.9.5
# - Tailwind CSS 3.4.17 + PostCSS + Autoprefixer
# - Radix UI: @radix-ui/react-dropdown-menu, @radix-ui/react-slot
# - Utility libraries: class-variance-authority, clsx, tailwind-merge
# - Testing libraries: @testing-library/react, jest, web-vitals

# 8. Troubleshooting Commands:
# Clear cache: rm -rf node_modules package-lock.json && npm install
# Kill port: npx kill-port 3000 (or use setup script)
# Check versions: node --version && npm --version
# Deploy to Vercel: vercel --yes
